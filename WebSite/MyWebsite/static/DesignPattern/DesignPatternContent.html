Abstract Factory Pattern: is way to instantiate families of related objects without using "new" keyword. In the pattern usage, an abstract class is extended by mutiple other abstract classes(Factory). These 2nd level of abstract classes are factory classes that is extended by concrete classes. And the only job of 2nd level abstract class is to instantiate and return correct concetre class.
To use factory class, the client code invoke the factory method of first level of Abstract class, which returns appropriate child Factory class reference. The factory method call on the child-factory class return the actual concrete object.


Code Sample
 
                Step-1
            public interface Shape {
            void draw();
            }
            
                Step-2
            public class Rectangle implements Shape {
               @Override
               public void draw() {
                  System.out.println("Inside Rectangle::draw() method.");
               }
            }
            public class Square implements Shape {
                @Override
                public void draw() {
                    System.out.println("Inside Square::draw() method.");
                }
            }
            public class Circle implements Shape {
               @Override
               public void draw() {
                  System.out.println("Inside Circle::draw() method.");
               }
            }
                Step-2
         public class  AbstractFactoryPatternDemo {    
        public static void main(String[] args) {
              //get shape factory
              AbstractFactory shapeFactory = FactoryProducer.getFactory("SHAPE");
              //get an object of Shape Circle
              Shape shape1 = shapeFactory.getShape("CIRCLE");
              //call draw method of Shape Circle
              shape1.draw();
              //get an object of Shape Rectangle
              Shape shape2 = shapeFactory.getShape("RECTANGLE");
              //call draw method of Shape Rectangle
              shape2.draw();
      
              //get an object of Shape Square 
              Shape shape3 = shapeFactory.getShape("SQUARE");
              //call draw method of Shape Square
              shape3.draw();
              //get color factory
              AbstractFactory colorFactory = FactoryProducer.getFactory("COLOR");
              //get an object of Color Red
              Color color1 = colorFactory.getColor("RED");
              //call fill method of Red
              color1.fill();
              //get an object of Color Green
              Color color2 = colorFactory.getColor("Green");
              //call fill method of Green
              color2.fill();
              //get an object of Color Blue
              Color color3 = colorFactory.getColor("BLUE");
              //call fill method of Color Blue
              color3.fill();
           }
        }
