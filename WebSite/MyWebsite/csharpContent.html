Code written with a language compiler that targets the runtime .NET CLR (Common Language Runtime) is called Managed code
The Common Language Runtime (CLR), the virtual machine component of Microsoft's .NET framework, manages the execution of .NET programs. Language Compiler compiles .NET Code into MSIL (MicroSoft Intermidiate Lanaguage). JIT (Just In Time) Compiler converts MSIL Code into Machine code which then gets executes.
